CodeCourt

Static coding benchmarks are easy to memorize. CodeCourt replaces them with seeded, adversarial tasks that expose brittle reasoning under hidden tests. The proof is not one lucky sample. It is a learning story: failure first, reward movement during training, and a concrete boundary-condition capability that improves afterward.
